# CSE422 Computer Graphics Lab

This repository contains a collection of computer graphics algorithms and projects implemented as part of the CSE422 Computer Graphics Lab course. Each project demonstrates a specific algorithm or concept in computer graphics, showcasing the use of C++ and Python for graphical computations and visualizations.

## Repository Structure

### 1. Bresenhams_Algorithm
- **Description**: Implementation of Bresenham's Line Drawing Algorithm, which efficiently determines the points of a rasterized line between two endpoints.
- **Key Files**: `main.cpp` (algorithm implementation), project files.

### 2. Circle_Algorithm
- **Description**: Demonstrates a circle drawing algorithm using incremental calculations to plot points along the circumference.
- **Key Files**: `main.cpp` (algorithm implementation), project files.

### 3. Colored_Rectangle
- **Description**: Renders a rectangle filled with a solid color, showcasing basic 2D rendering techniques.
- **Key Files**: `main.cpp` (rendering logic), project files.

### 4. Colorful_star
- **Description**: Creates a colorful star using graphical primitives and color blending techniques.
- **Key Files**: `main.cpp` (star rendering logic), project files.

### 5. DDA_Algorithm
- **Description**: Implements the Digital Differential Analyzer (DDA) algorithm for line drawing, which uses floating-point arithmetic for smooth rasterization.
- **Key Files**: `main.cpp` (algorithm implementation), project files.

### 6. Mid_Point_Circle_Algorithm
- **Description**: Implements the Midpoint Circle Drawing Algorithm, an efficient rasterization technique for drawing circles.
- **Key Files**: `main.cpp` (algorithm implementation), project files.

### 7. Two-dimensional_Object_Animation
- **Description**: Demonstrates 2D object animation, including movement and transformations over time.
- **Key Files**: `main.cpp` (animation logic), project files.

### 8. Two-dimensional_Object_Animation_Spinning_Sound
- **Description**: Adds spinning and sound effects to animated 2D objects, combining visual and auditory elements.
- **Key Files**: `main.cpp` (animation and sound integration), project files.

### 9. Two-dimensional_Object_Rotation
- **Description**: Implements rotation transformations for 2D objects around a pivot point.
- **Key Files**: `main.cpp` (rotation logic), project files.

### 10. Two-dimensional_Object_Tranaslation
- **Description**: Demonstrates translation transformations for 2D objects, moving them across the screen.
- **Key Files**: `main.cpp` (translation logic), project files.

### 11. _Fianl_Lab_Report
- **Description**: Contains the final lab report and supplementary resources, including input/output images and a Python implementation of the Midpoint Circle Algorithm.
- **Key Files**: `FianlLabReport.pdf` (report), `Midpoint_Circle.py` (Python implementation), `input.PNG`, `output.PNG`.

## How to Run
1. Navigate to the respective project folder.
2. Open the `.cbp` file in Code::Blocks or your preferred IDE.
3. Build and run the project.

For Python scripts, ensure Python is installed and run the script using:
```bash
python <script_name>.py
```
